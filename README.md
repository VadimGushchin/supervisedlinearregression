# Ml Project: Supervised Learning с использованием LinearRegression

Данный проект был выполнен мной в рамках обучения в Школе 21 от СберБанка. 

Целью задания было ознакомиться с классической моделью машинного обучения - Линейной Регрессией, реализовать кастомный класс с различными оптимизаторами(градиентный спуск, стохастический градиентный спуск, мини-батч спуск, аналитическое решение), кастомные регуляризации и нормализации, а после все это сравнить с эталонами из библиотеки sklearn. 

Дополнительно, после выполнения основного задания, был реализован класс my_featureeng.py, реализующий методы для добавления новых признаков в модель, позволивший "пробить" потолок значений метрик и значительно повысить $R^2$ score, MAE и RMSE.

По проекту работа проходила с датасетом train.json, взятым с сайта Kaggle, а конкретно из соревнования Two Sigma Connect: Rental Listing Inquiries. В рамках выполнения задания, таргетом являлась цена или 'price'.

## Описание структуры репозитория

Все файлы проекта лежат в /src

Внутри находятся .py файлы с реализованными классами, ML2.ipynb с основным отчетом по проекту и папка /data, из которой проект берет датасеты. Сами json файлы датасетов автоматически скачиваются с гугл диска, если в папке их еще нет.

Реализованные классы:

* **my_DataProcessing** - класс обработки данных. Содержит функции для очистки, преобразования и форматированния данных
* **my_featureeng** - класс для **Feature Engineering**. Создает новые признаки по заранее заданным параметрам, отбирает ненулевые.
* **my_graphs** - класс построения pyplot графиков, для переиспользования одинакового кода, опционален
* **my_GridParamsSearch** - класс поиска оптимальных параметров для моделей по сетке
* **my_LinearRegression** - основной класс проекта, реализация **Линейной Регрессии**
* **my_ModelAnalytics** - вспомогательный класс, анализирует метрики и выводит лучшую модель
* **my_Scalers** - реализация StarndardScaler и MinMaxScaler
