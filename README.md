# Ml Project: Supervised Learning с использованием LinearRegression

Данный проект был выполнен мной в рамках обучения в Школе 21 от СберБанка. Целью задания было ознакомиться с классической моделью машинного обучения - Линейной Регрессией, реализовать кастомный класс с различными оптимизаторами(градиентный спуск, стохастический градиентный спуск, мини-батч спуск, аналитическое решение), кастомные регуляризации и нормализации, а после все это сравнить с эталонами из библиотеки sklearn. Дополнительно мной, после выполнения основного задания, был реализован класс my_featureeng.py, реализующий методы для добавления новых признаков в модель, позволивший "пробить" потолок значений метрик и значительно повысить R^2 score, MAE и RMSE.
